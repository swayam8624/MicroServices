## 4. Complete Workflow Test

### 4.1 Create a New Product

#### Request:
curl -X POST http://localhost:9081/api/products -H "Content-Type: application/json" -d '{"name": "Tablet", "description": "10-inch tablet", "price": 500.0, "quantity": 15}'

#### Response:
{
  "id": 4,
  "name": "Tablet",
  "description": "10-inch tablet",
  "price": 500.0,
  "quantity": 15
}

### 4.2 Create an Order for the New Product

#### Request:
curl -X POST http://localhost:9082/api/orders -H "Content-Type: application/json" -d '{"customerId": 2, "items": [{"productId": 4, "quantity": 1}]}'

#### Response:
{
  "id": 2,
  "customerId": 2,
  "items": [
    {
      "productId": 4,
      "productName": "Tablet",
      "quantity": 1,
      "price": 500.0
    }
  ],
  "totalAmount": 500.0,
  "status": "CREATED"
}

### 4.3 Process Payment for the Order

#### Request:
curl -X POST http://localhost:9083/api/payments -H "Content-Type: application/json" -d '{"orderId": 2, "paymentMethod": "DEBIT_CARD"}'

#### Response:
{
  "id": null,
  "orderId": 2,
  "amount": 500.0,
  "status": "COMPLETED",
  "paymentMethod": "DEBIT_CARD"
